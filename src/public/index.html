Quick Example

curl -X POST https://your-domain.com/api/scrape/listings \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "city": "oakland",
    "state": "ca",
    "filters": {
      "maxPrice": 3000,
      "minBeds": 2,
      "sortBy": "price",
      "limit": 20
    }
  }'
